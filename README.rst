================
Ploneconf policy
================

A policy package for Ploneconf site

Features
--------

Custom content-types:

- Person (a trainer or speaker)
- Talk
- Training
- Time Slot
- Scheduled Talk


Installation
------------

Install ploneconf.policy by adding it to your buildout::

    [buildout]

    ...

    eggs =
        ploneconf.policy


and then running ``bin/buildout``
